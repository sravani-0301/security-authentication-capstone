# Security Authentication Capstone

## Project Overview

This project demonstrates secure coding practices for the SafeVault application.

## Features

### Input Validation
Validates user input and rejects malicious characters.

### SQL Injection Prevention
Uses parameterized query concepts to prevent SQL injection.

### XSS Protection
Sanitizes user input before processing.

### Authentication
Verifies user credentials before login.

### Authorization
Implements role-based access control (RBAC).

### Security Testing
Includes tests for validation, authentication, authorization, and XSS prevention.

## Activities Completed

- Writing Secure Code Using Microsoft Copilot
- Implementing Authentication and Authorization Using Microsoft Copilot
- Debugging and Resolving Security Issues Using Microsoft Copilot
